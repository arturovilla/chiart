# chiart.gallery

A virtual scrapbook for Chiamaka — built with love for our anniversary.

## Tech Stack

- [Next.js](https://nextjs.org) (App Router)
- Deployed on [Vercel](https://vercel.com)
- Custom domain: [chiart.gallery](https://chiart.gallery)

## Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view locally.
